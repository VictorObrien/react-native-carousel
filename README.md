## :running: Running the project

1. Clone this project:

```sh
  $ git clone https://github.com/VictorObrien/react-native-carousel.git
```

2. Install the dependencies

```sh
  yarn
```
or

```sh
  npm install
```

3. Start


```sh
  yarn start
```
or

```sh
  npm start
```
